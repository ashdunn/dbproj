# dbproj

This application allows you to use speech recognition to execute queries on an Instacart database.

## Getting Started

clone the project:
```
git@github.com:ashdunn/dbproj
```

```
cd dbproj
```

```
python3 alexa.py
```

### Prerequisites

python3
pyaudio
tkinter
speech_recognition
pillow
